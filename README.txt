Sequential Jobs
Sequential Jobs is an application designed to help users manage their job sequencing process. With Sequential Jobs, users can create a list of job tasks, define the order in which the tasks should be performed, and track the progress of each task. The app provides a user-friendly interface to make managing job sequencing easier and more efficient.

Features
Create job tasks and define the order in which they should be performed
Set deadlines for each job task
Track the progress of each task
Receive email notifications when a task is completed or overdue
View a Gantt chart of the job sequence to visualize the schedule
